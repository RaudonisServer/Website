Rules

  1. These rules apply to both the Minecraft server and the Signal chat.
  2. Moderators are not required to justify their actions.
  3. Be kind to others and use common sense.
  4. Griefing of any kind is prohibited.
  5. Minecraft modifications are only allowed if they do not provide an unfair advantage (e.g. auto-clicking, etc.).

Strictly prohibited client-side modifications/resource packs:

  1. X-Ray
  2. Meteor Client
  3. Wurst Client
  4. And anything similar

Any of these modifications/resource packs could result in a permanent ban if used repeatedly.

Please report hackers to a moderator/developer immediately if you notice them.
